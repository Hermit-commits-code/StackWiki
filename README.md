# StackWiki

StackWiki is a private developer wiki for Markdown-based technical notes, tags, search, and C#/.NET learning references.

## Purpose

This project is being built to support my C# 14 and .NET 10 learning path while also becoming a real portfolio project.

## Planned Stack

- C# 14
- .NET 10
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- React
- Docker
- xUnit

## MVP Goals

- Create notes
- Store Markdown content
- Add tags
- Search notes
- View notes in a clean wiki interface

## Version

Current target: v1.0.0 initial scaffold
